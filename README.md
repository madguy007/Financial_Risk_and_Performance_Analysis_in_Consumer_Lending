# 🚀 FinTech Risk Analytics System

## 📌 Overview

This project builds a **complete end-to-end data analytics system** for a FinTech lending platform, covering:

* Business understanding
* Data modeling
* SQL-based ETL pipelines
* Excel-based analysis & dashboarding
* Power BI visualization
* n8n + GenAI automation

---

## 🧠 Problem Statement

In consumer lending, financial institutions must balance:

* 📈 Loan growth
* ⚠️ Risk management

However:

* Risk teams don’t check dashboards daily
* Decision explanations are often unclear

👉 This project solves both by combining **analytics + automation + AI**

---

## 🔄 End-to-End Pipeline

Business Understanding → ETL → Analysis → Dashboard → Automation

## 📈 Key Insights

* High DTI and low employment length → higher default risk
* Certain regions show concentrated default patterns
* Loan grade strongly correlates with repayment behavior
* Improved strategies increased loan adoption without major risk increase
