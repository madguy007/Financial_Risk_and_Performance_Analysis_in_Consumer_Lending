# 📊 Power BI Dashboard — Loan Risk & Performance

## 📌 Objective

To build an **interactive dashboard** that provides insights into loan performance, risk distribution, and borrower behavior.

---

## 📂 Files

* 📊 Power BI File:
  [Open Dashboard](./dashboard.pbix)

---

## 📊 Dashboard Overview

The dashboard provides a comprehensive view of:

* Loan distribution across regions
* Risk segmentation by loan grade
* Default rate trends
* Borrower income vs loan analysis

## 📈 Key Metrics

* Total loans issued
* Average loan amount
* Default rate
* Interest rate trends

## 🎯 Key Insights

* High-risk grades show higher default rates
* Medium-income segment contributes significantly to loan volume
* Certain regions exhibit higher default concentration

## 🧠 Features

* Interactive filters (year, region, grade)
* Drill-down capability
* KPI cards for quick insights
* Dynamic visualizations

## 🛠 Tools Used
